# Detection of Obfuscated Jailbreak and Prompt Injection Attacks in Large Language Models

This project focuses on detecting obfuscated jailbreak prompts that attempt to bypass safety mechanisms in Large Language Models.

## Objectives
- Study existing jailbreak techniques
- Review AI Safety literature
- Build a dataset
- Evaluate detection approaches
- Develop a research paper
